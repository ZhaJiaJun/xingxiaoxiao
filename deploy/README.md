# 星消消 ✨

消除烦恼，快乐无限！

一款 HTML5 消除类休闲小游戏——点击同色相邻星星完成消除，达成目标分数即可晋级。

## 🎮 怎么玩

- 点击一组 **相邻（上下左右）** 的 **同色星星**（≥2 颗）
- 消除分 = n² × 5
- 剩余越少奖励越高（满分 2000）
- 无时间限制，随时暂停

## 🛠️ 道具

| 道具 | 价格 | 效果 |
|------|:--:|------|
| 🔨 锤子 | 20 积分 | 敲碎单个星星 |
| ➕ 十字锤 | 50 积分 | 消除十字区域 |
| 🔄 洗牌 | 120 积分 | 重新排列星星 |

## 🎨 特色

- 5 色高对比度方块 + 色盲图案模式
- 8bit 芯片音效
- 多存档独立积分，最高分全存档共享
- 跳关按钮（达标即显）
- 纯前端，双击即玩
